The Fortran90 source code is in GDF.FOR. and a the example program is GDF_EXAMPLE.FOR. The main LaTeX file is GDF.TEXm which references many other files in this directory..

Installation

The program GDF_EXAMPLE.F90 has several purposes. If this program is run without any command line arguments it will produce the GDF documentation files in Latex format. Compiling the file GDF.TEX with Latex and the converting GDF.DVI produces the postscript file GDF.PS. This file will normally be available in the same directory as the GDF source code.

If GDF_EXAMPLE is with a file name as a single command line argument it will print out the contents of an existing data file. This can be used to check that the files content is read correctly. For example a command such as gdf_example gt008742.fz will display the contents of the first few events in the data file.

The example program can also perform a GDF self-test, which writes a new data file, and then reads it to checks that the values read are as expected. The command is gdf_example test.fz test.fz. The first argument is the test file to be created, the second argument the name of the file to be read and checked.
